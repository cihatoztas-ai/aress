CAP-02 DN / inch → mm Dönüşüm Sistemi
YENİ▼
Dışarıdan DN veya inch formatında veri girildiğinde capToMM() fonksiyonu devreye girer. Supabase geçişinde tanımlar tablosundan okuyacak — şimdilik sabit map.
// capToMM() — DN, inch veya mm girişini dış çap mm'ye döndürür
function capToMM(cap) {
  if (DN_TO_MM[cap]) return DN_TO_MM[cap];
  if (INCH_TO_MM[cap]) return INCH_TO_MM[cap];
  var n = parseFloat(cap);
  if (!isNaN(n) && n > 0) return n; // zaten mm
  return null;
}

// formatCapMM() — gösterim formatı (114.3 → "114,3")
function formatCapMM(cap) {
  var mm = capToMM(cap);
  return mm % 1 === 0 ? String(mm) : String(mm).replace('.', ',');
}

K-01 JS Syntax Kontrolü — Her Sayfada Zorunlu
▼
Her sayfa tesliminden önce Node.js ile syntax kontrolü yapılır. Tarayıcıda test öncesi bu adım atlanamaz. Hata varsa teslim kabul edilmez.
Kontrol Komutu (terminal)
node -e "
  var fs=require('fs');
  var c=fs.readFileSync('sayfa.html','utf8');
  var s=c.indexOf("'use strict';"), e=c.lastIndexOf('</script>');
  try{new Function(c.slice(s,e));console.log('✅ JS OK');}
  catch(err){console.log('❌ HATA:',err.message);}
"
Tüm Sayfaları Toplu Tara
# app/ klasöründen çalıştır
for f in *.html; do
  node -e "var fs=require('fs'),c=fs.readFileSync('$f','utf8'),s=c.indexOf("'use strict';"),e=c.lastIndexOf('</script>');if(s<0){console.log('⏭ JS yok: $f');process.exit(0);}try{new Function(c.slice(s,e));console.log('✅ $f');}catch(err){console.log('❌ $f: '+err.message);}"
done
⚠️ Sık karşılaşılan hatalar:
• JS string içinde tek tırnak → font-family:'Barlow' gibi CSS kaldır veya kaçır
• Python triple-quote içinde ' → çıktıda ' olur, JS string kırılır
• QRCode, Swiper gibi CDN kütüphaneleri yüklenmeden çağrılırsa → ReferenceError
